## how to run locally

cloen repo

```git clone https://github.com/kon-rad/ethdenver2022```

cd into folder

```cd ethdenver2022```

install dependencies

```yarn```

run local hardhat node

```yarn local```

open a new tab

compile and deploy smart contracts

```yarn deploy:local```

open a new tab

run web app

```yarn dev```
